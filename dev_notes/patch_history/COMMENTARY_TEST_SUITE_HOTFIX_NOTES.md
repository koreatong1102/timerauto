# COMMENTARY_TEST_SUITE_HOTFIX_NOTES

## 목적
설정의 테스트탭에서 자동해설 전체 흐름을 직접 들어볼 수 있도록 테스트 버튼을 추가했습니다.

## 추가된 테스트
- 자동해설 종합 테스트
  - 경기 시작, 소강상태, RFC 짧은 받아치기, 잽/카운터, 데미지, 다운, 라운드 요약, 경기 종료 요약을 순서대로 재생합니다.
- 다운 멘트 테스트
  - 1다운, 2다운, 3다운, 재개/종료 멘트 흐름을 확인합니다.
- 요약 멘트 테스트
  - 휴식시간 라운드 요약과 경기 종료 요약 톤을 확인합니다.
- 해설 테스트 중지
  - 진행 중인 테스트 TTS와 예약된 테스트 큐를 중지합니다.

## 동작 방식
- 테스트 스크립트 큐를 사용해 캐스터/해설자 음성이 순서대로 재생됩니다.
- 이전 문장이 아직 재생 중이면 다음 문장을 기다립니다.
- Edge TTS 생성/재생 지연으로 문장이 씹히는 것을 줄이기 위해 busy 상태를 확인합니다.
- 테스트 버튼은 테스트탭에 배치했고, 사운드 설정 그룹에도 동일 기능 버튼을 유지했습니다.

## 주의
- 실제 SpectatorLog 판정 로직을 강제로 조작하는 기능이 아니라, TTS 톤/길이/역할 분리/요약 발음을 빠르게 들어보는 테스트입니다.
- OBS 브라우저 오버레이 테스트는 기존 HUD/이펙트 테스트 버튼을 사용하면 됩니다.
