# TimerAuto 간단 사용 설명서

이 문서는 처음 사용하는 사람도 바로 따라할 수 있게 핵심만 정리한 안내서입니다.

## 1. 프로그램 실행

1. `timerauto.exe`를 실행합니다.
2. 설정 창에서 먼저 ROI와 트리거를 맞춥니다.

## 2. 꼭 해야 하는 초기 설정

### 2-1. ROI 지정

ROI는 OCR이 글자를 읽는 화면 영역입니다.

우선 아래 2개를 정확히 잡으세요.
- 라운드/초 표시 ROI
- 시간 ROI

정확하지 않으면 타이머와 안내 문구가 틀어집니다.

### 2-2. 트리거/액션 확인

- `on_trigger` 액션에 필요한 동작(마우스 이동, TTS 등)이 들어있는지 확인합니다.
- 테스트 버튼으로 한 번 재생해 보고 실제 음성이 나오는지 확인합니다.

## 3. 방송 동기화 버튼(중요)

타이머 시작 시간과 실제 방송 시작 시간이 다르면 챕터 시간도 밀립니다.

오버레이 상단 메뉴의 `방송시작동기화` 버튼으로 기준 시간을 맞추세요.

권장 순서:
1. 방송 시작 직후 `방송시작동기화` 클릭
2. 트리거 동작 시작
3. 종료 후 챕터 로그 확인

## 4. 챕터 로그 저장

- 챕터 로그는 `jsonl` + `txt` 둘 다 저장됩니다.
- 저장 폴더는 설정에서 지정할 수 있습니다.
- 프로그램 종료 시 자동 저장도 동작합니다.

## 5. TTS(매치업 안내) 사용 팁

- 닉네임이 등록되어 있으면 닉네임을 읽습니다.
- 닉네임이 없으면 ID를 읽습니다.
- 음성이 안 나오면 로그에서 `TTS_` 항목을 먼저 확인하세요.

## 6. 선수 데이터 내보내기

플레이어 탭에서 선수 목록을 TXT로 내보낼 수 있습니다.

형식: `닉네임 아이디`

## 7. 배포 전 체크리스트

1. 트리거 3회 이상 연속 테스트
2. TTS 반복 재생 테스트
3. 챕터 로그 파일 생성 확인
4. 설정창/오버레이 UI 겹침 문제 없는지 확인

## 8. 자주 발생하는 문제

### Q. TTS가 안 나와요.
- 로그에 `Wrong version`이 반복되면 TTS 엔진 초기화 실패입니다.
- 최신 빌드로 다시 테스트하세요.

### Q. 설치파일이 안 만들어져요.
- `iscc not found` 경고가 뜨면 Inno Setup이 없거나 PATH 미설정 상태입니다.
