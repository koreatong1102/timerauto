# RFC Witty Duo Commentary Patch

기준 소스: timer_best_commentary_break_budget_source.zip

## 목적
- 실시간 중계가 더 자연스럽고 가볍게 들리도록 캐스터-해설자 짧은 받아치기 구조를 추가했다.
- 기존 브레이크 시간예산형 요약, 경기 종료 요약, 게이지 기반 정확도, 캐스터/해설자 독립 재생 정책은 유지했다.

## 핵심 변경
- 안전한 fight 상태에서 캐스터가 짧게 상황을 말한 뒤, 해설자가 약 1초 뒤 짧은 한 줄로 받아친다.
- 위험 상황, 다운, 스턴, TKO/KO, 게이지 위험, 휴식 요약/경기 종료 요약 중에는 받아치기 유머를 차단한다.
- "계산서", "문", "초대장", "영수증"류의 비유형 유머는 넣지 않았다.
- 바로 이해되는 리액션형 문장만 사용한다.

## 적용된 유머 카테고리
- 소강상태: 둘 다 먼저 가긴 싫죠 / 서로 오라고만 합니다 / 아직 간만 봅니다
- 라운드 막판: 막판엔 한 방입니다 / 여기서 실수하면 큽니다
- 잽/앞손: 앞손이 오늘 바쁩니다 / 잽 출근률 높습니다
- 미스 확정 문맥: 폼은 멋졌습니다 / 공기는 맞았습니다
- 압박: 이러면 피곤합니다 / 수비가 바빠집니다
- VR 복싱 느낌: 팔은 많이 나갔습니다 / 운동량은 확실합니다

## 제한 장치
- quiet 모드에서는 비활성화.
- standard 모드는 라운드당 최대 4회.
- active 모드는 라운드당 최대 6회.
- 같은 카테고리 반복 쿨다운 및 최근 문장 반복 방지 적용.
- 한 문장은 28자 이하 후보만 선택.

## 요약 변경
- 휴식시간 라운드 요약에 너무 과한 드립 대신 짧고 자연스러운 표현만 소량 추가했다.
- 경기 종료 요약에는 점수/승자 설명을 해치지 않는 선에서 한 줄 리액션 후보만 추가했다.
