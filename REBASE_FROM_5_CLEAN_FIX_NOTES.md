# 5.zip 기준 재정리 핫픽스

- 5.zip을 기준으로 다시 작업했습니다.
- 이전 꼬인 설정창 방어 패치들을 무작정 가져오지 않고, 필요한 것만 최소 적용했습니다.
- 설정창 진입 안정화: SettingsDialog 생성 시 Qt Multimedia 플레이어를 만들지 않고, 테스트 버튼을 누를 때만 지연 생성합니다.
- UI 투명도: 창 전체가 아니라 QML 검은 배경 Rectangle만 투명해집니다. 텍스트/초상화/아이디/버튼은 불투명하게 유지됩니다.
- 초상화 우선순위: 기본은 로그 초상화 우선이며, 이 모드에서는 프로필 사진으로 fallback하지 않습니다. 설정에서 프로필 우선으로 바꾸면 프로필 사진을 먼저 쓰고, 프로필 사진이 없을 때 로그 초상화를 사용합니다.
- build_release 기본 config에 overlay_ui_bg_opacity, portrait_source_priority를 포함했습니다.
