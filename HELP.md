# TimerAuto 사용 가이드

## 빠른 시작

1. `timerauto.exe` 실행
2. ROI(라운드/시간) 지정
3. 트리거 규칙 확인
4. 액션 테스트로 TTS 확인
5. 오버레이에서 `방송시작동기화` 클릭
6. 감지 시작

## 핵심 기능 설명

### 1) 방송시작동기화

- 용도: 방송 기준 시간과 프로그램 기준 시간을 맞춰 챕터 로그 시간을 정확하게 맞춤
- 위치: 오버레이 상단 메뉴
- 권장: 방송 시작 직후 1회 클릭

### 2) 트리거 + 액션

- 트리거: 화면 조건이 맞는 순간 감지
- 액션: 감지 후 실행할 동작(마우스, 키 입력, TTS, 타이머 제어 등)
- 팁: 실제 경기 화면으로 최소 3회 반복 테스트

### 3) 매치업 TTS

- 닉네임 등록 시 닉네임 우선 읽기
- 미등록 시 ID 읽기
- 반복 횟수/속도/문구는 액션 설정에서 조절

### 4) 챕터 로그

- 출력 형식: `txt`, `jsonl`
- 저장 폴더: 설정에서 지정 가능
- 프로그램 종료 시 자동 저장 지원

### 5) 선수 목록 내보내기

- 플레이어 탭에서 TXT 내보내기 지원
- 출력 형식: `닉네임 아이디`

## 권장 설정 순서

1. ROI 정확도 먼저 확보
2. 트리거 색상/허용 오차 조정
3. 액션 순서(딜레이 포함) 정리
4. TTS 테스트
5. 방송시작동기화 후 실전 시작

## 문제 해결

### TTS가 안 나오는 경우

1. 로그에서 `TTS_WORKER`, `TTS_ENQUEUE`, `TTS_DEQUEUE` 존재 여부 확인
2. `Wrong version` 관련 오류가 있으면 최신 빌드로 재실행
3. 테스트 액션에서 먼저 정상 재생 확인 후 실전 트리거 적용

### 설치파일 빌드가 안 되는 경우

- `iscc not found` 경고: Inno Setup 컴파일러가 PATH에 없음
- 조치: Inno Setup 설치 후 빌드 재실행
