# TTS WORDING HOTFIX

- TTS가 `누적됩니다`를 어색하게 읽는 문제를 피하기 위해 해설 출력 문구의 `누적` 계열 표현을 `쌓입니다`, `쌓이고 있습니다`, `쌓인 데미지` 계열로 정리했습니다.
- TTS가 `게이지`를 어색하게 읽는 문제를 피하기 위해 해설 출력 문구의 `게이지` 계열 표현을 `데미지`, `체력 흐름`, `체력 여유`, `받은 데미지` 계열로 정리했습니다.
- UI 버튼/내부 변수/내부 판정 로직의 용어는 기능 의미가 바뀌지 않도록 유지했습니다.
- 기존 다운 상태 머신, witty duo, 브레이크 시간예산 요약, 스코어카드 요약, 게이지 기반 내부 판정은 유지했습니다.
