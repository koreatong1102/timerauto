# Stage36 Sprite Hit FX
- Adds a self-made transparent PNG sprite atlas for Tekken-like punch impact bursts.
- Browser overlay preloads the sprite through inline CSS data URI and reuses the existing impact node pool.
- Settings include: explosion sprite ON/OFF and old ring/glow ON/OFF.
- Adds settings test button: 폭발 피격 테스트.
- Keeps fast path, latency logs, damage threshold, duration, size, opacity, glow, fill and color-picker settings.
