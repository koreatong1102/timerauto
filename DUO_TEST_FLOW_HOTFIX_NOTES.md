# DUO TEST FLOW HOTFIX

- 설정 > 테스트탭에 `듀오 해설 테스트` 버튼을 추가했다.
- 기존 테스트 스크립트 큐는 캐스터/해설자 둘 다 끝날 때까지 기다려서 듀오처럼 들리지 않았다.
- 테스트 스크립트에 `followup_text`, `followup_delay_ms`, `followup_role`을 추가했다.
- 캐스터 테스트 문장 재생 후 약 0.9~1.0초 뒤 해설자 받아치기 멘트가 별도 역할 플레이어로 들어간다.
- 위험/다운 구간은 듀오 드립 없이 진지한 멘트만 나온다.
- 실제 자동해설의 witty duo 후속 슬롯은 유지하고, 테스트탭에서도 그 흐름을 귀로 확인할 수 있게 했다.
